# plan-generator
## RUNNING
> $ go run main.go

# Dependency management tool (dep)
``` https://golang.github.io/dep/ ```

## Installation
### MAC OS
> $ brew install dep\
> $ brew upgrade dep
### Install From Source
> go get -d -u github.com/golang/dep


## Install dependencies
Go to project folder
> cd $(go env GOPATH)/src/github.com/gy-kim/plan-generator

Run dep
> dep ensure
